# 📬 Formulário de Contato

Uma página de contato completa com informações da empresa à esquerda e formulário de envio de mensagem à direita — desenvolvida com HTML e CSS puro.

---

## 📸 Preview

> Card dividido em duas colunas: à esquerda, informações de contato (Endereço, Telefone e Email) com ícones roxos. À direita, formulário com título "Envie uma mensagem", campos de nome, e-mail e mensagem, e botão **Enviar Agora** em roxo/azul.

---

## 🚀 Tecnologias

- **HTML5**
- **CSS3**

---

## 📁 Estrutura do Projeto

```
formulario-de-contato/
├── index.html
└── style.css
```

---

## ⚙️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/formulario-de-contato.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd formulario-de-contato
   ```

3. Abra o `index.html` no navegador ou via **Live Server** no VS Code:
   - Acesse em: `http://127.0.0.1:5500`

---

## 🎯 Funcionalidades

- Layout em duas colunas: informações de contato + formulário
- Três campos de input: nome, e-mail e mensagem (textarea)
- Botão **Enviar Agora** estilizado
- Informações de contato com ícones ilustrativos

## 🎨 Design

- Fundo da página: azul claro/ciano suave
- Card principal: branco com bordas arredondadas e sombra
- Coluna esquerda: fundo levemente acinzentado com ícones roxos
- Coluna direita: fundo branco com campos de input cinza claro
- Botão: roxo/azul (`#5c5be5` aprox.) com texto branco
- Tipografia: sans-serif, título em roxo escuro
